# Plan: Deploy to GitHub Pages via gh-pages package

## Context
Switch from GitHub Actions to `gh-pages` npm package for manual deployment.

## Steps

### 1. Remove GitHub Actions workflow
Delete `.github/workflows/deploy.yml`

### 2. Install gh-pages
```bash
npm install --save-dev gh-pages
```

### 3. Add deploy script to `package.json`
```json
"deploy": "gh-pages -d dist"
```

### 4. Run deployment
```bash
npm run build
npm run deploy
```

### 5. Configure GitHub Pages
Go to **Settings > Pages > Source** and select **Deploy from a branch** > **gh-pages** branch.
